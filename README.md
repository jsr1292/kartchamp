# 🏎️ KartChamp

Go-kart championship tracker for friends and amateur leagues.

## Stack
- SvelteKit + TypeScript
- Tailwind CSS
- Drizzle ORM + PostgreSQL (Neon)
- Vercel deployment

## Dev

```bash
npm install
npm run dev
```

## Database

```bash
# Generate migration
npx drizzle-kit generate

# Push schema
npx drizzle-kit push
```
